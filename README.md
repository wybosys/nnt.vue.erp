# nnt.vue.erp

> 基于nnt.vue开发ERP类标准项目

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# 生成单站点模式的route文件
npm run routes

# 生成多站点模式的route文件
npm run sites
```
